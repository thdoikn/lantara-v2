# Sektor Pendidikan — 9 Izin (SK-complete)

Ground truth: **SK Standar Pelayanan Perizinan Sektor Pendidikan di Ibu Kota Nusantara** (Keputusan Kepala Otorita IKN, 2026). Each fixture reproduces the SK's Persyaratan → `doc_requirements`, Sistem/Mekanisme → `stages`, Jangka Waktu → `sla_days`, Produk → `product_name`, Dasar Hukum → `legal_basis`, and pengaduan channels → `complaint_info`.

Pengampu: Direktorat Pelayanan Dasar (Dit. Yandas).

## Workflow (two-bidang, 5 stages — per SK swimlane)
`submit → verifikasi-berkas (Bidang Perizinan) → verifikasi-data (Bidang Pelayanan Dasar, + kunjungan lapangan) → penerbitan (Kepala Otorita) → pengambilan`.
